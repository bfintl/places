ISO 3166-2:IN

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:IN is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to India.

Each two-part code consists of the ISO 3166-1 code for India (IN) and a 2-character sub-code, hyphenated together.
