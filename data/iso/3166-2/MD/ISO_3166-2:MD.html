ISO 3166-2:MD

From Wikipedia, the free encyclopedia

Jump to: navigation, search

|ISO 3166-2:MD is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Moldova.

Each two-part code consists of the ISO 3166-1 code for Moldova (MD) and a 2-character-alphabetic sub-code, hyphenated together.
