ISO 3166-2:VE

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:VE is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Venezuela. Each two-part code consists of the ISO 3166-1 code for Venezuela (VE) and a single character, hyphenated together.
