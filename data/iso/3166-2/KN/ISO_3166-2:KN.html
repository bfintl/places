ISO 3166-2:KN

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:KN is an ISO standard which defines geocodes. It is the subset of ISO 3166-2 that applies to Saint Kitts and Nevis. Each two-part code consists of the ISO 3166-1 code for Saint Kitts and Nevis (KN) and a two-digit number.
