ISO 3166-2:PH

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:PH is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to the Philippines. It covers the 79 provinces (lalawigan) and 16 regions (rehiyon).

In 2002, the Southern Tagalog region was divided into the CALABARZON and MIMAROPA regions. As of 2005, ISO 3166-2:PH has not yet been modified to reflect those changes.

Each two-part code consists of the ISO 3166-1 country code for the Philippines and a 3-character subcode, hyphenated together.

[edit] Newsletters

The newsletter ISO 3166-2:2002-05-21 announced changes to ISO 3166-2:PH. This includes the addition of one region and six provinces, as well as minor spelling corrections.
