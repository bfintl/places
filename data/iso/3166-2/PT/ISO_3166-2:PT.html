ISO 3166-2:PT

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:PT is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Portugal.

The first part of the code is the ISO 3166-1 country code PT for Portugal, the second part is two-digit-numeric (with leading zeros).
