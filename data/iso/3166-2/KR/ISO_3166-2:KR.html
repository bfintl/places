ISO 3166-2:KR

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:KR is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to South Korea. It assigns codes to 1 special city, 6 metropolitan cities, and 9 provinces.

Each two-part code consists of the ISO 3166-1 code for South Korea (KR) and 2 digits, hyphenated together.
