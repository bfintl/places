ISO 3166-2:GR

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:GR is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Greece.

Each two-part code consists of the ISO 3166-1 code for Greece (GR) and a 2-alphanumeric-character subcode, hyphenated together.
