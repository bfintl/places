ISO 3166-2:KH

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:KH is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Cambodia. Each two-part code consists of the ISO 3166-1 code for Cambodia (KH) followed by a hyphen and either 1 or 2 digits.
