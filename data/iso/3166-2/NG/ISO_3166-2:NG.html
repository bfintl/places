ISO 3166-2:NG

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:NG is an International Organization for Standardization standard which defines a geocode: it is the subset of ISO 3166-2 which applies to Nigeria. Each code consists of the ISO 3166-1 code NG and a hyphen followed by 2 letters for each of the 36 states of the federation and the federal capital territory (FCT Abuja).
