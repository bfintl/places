ISO 3166-2:CN

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:CN is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to the People's Republic of China.

Each code consists of CN (the ISO 3166-1 2-character code for the PRC) followed by a 2-digit numeric code.
