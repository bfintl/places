ISO 3166-2:SI

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:SI is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Slovenia. It covers 193 Občine (municipalities), 11 of which have "urban status".

Each two-part code consists of the ISO 3166-1 code for Slovenia (SI) and a 3-character-numeric subcode, hyphenated together.
