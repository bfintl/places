ISO 3166-2:IT

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:IT is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Italy. It covers 110 provinces (Aosta Valley is a region, but considered as province). Each province has a 2-letter code, that is also used for car signs. Together with the ISO 3166-1 code IT for Italy they form the ISO 3166-2 codes.

Note: In 1999, the former Province Forlì / FO was renamed as Province Forlì-Cesena, commonly coded with FC. In the same year, the common coding for the Province Pesaro-Urbino has become PU instead of PS. However, ISO 3166-2 still sticks to the old codings.

[edit] Newsletters

  • ISO 3166-2:2000-06-21
  • ISO 3166-2:2007-04-17 - For Italy, it modified four codes:
    • IT-CI* Carbonia-Iglesias 88
    • IT-VS* Medio Campidano 88
    • IT-OG* Ogliastra 88
    • IT-OT* Olbia-Tempio 88
  • ISO 3166-2:2007 (second edition) - For Italy, it modified two codes:
    • IT-MA* Medio Campidano 88
    • IT-OL* Olbia-Tempio 88
