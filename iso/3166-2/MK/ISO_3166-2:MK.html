ISO 3166-2:MK

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:MK is an ISO standard which defines geocodes. It is the subset of ISO 3166-2 that applies to the Republic of Macedonia. Each two-part code consists of the ISO 3166-1 code for the Republic of Macedonia (MK) and a two-digit number.

[edit] Newsletters

  • ISO 3166-2:2007-04-17 - For the Republic of Macedonia, this introduced codes for the subdivisions.
  • ISO 3166-2:2007-11-28 - For the Republic of Macedonia, replaced the codes with numeric codes.
