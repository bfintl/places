ISO 3166-2:RO

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:RO is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Romania.

Each two-part code consists of the ISO 3166-1 code for Romania (RO) and a 1 or 2-character alphabetic sub-code, hyphenated together.
