ISO 3166-2:IR

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:IR is an ISO standard which defines geocodes. It is the subset of ISO 3166-2 that applies to Iran. Each two-part code consists of the ISO 3166-1 code for Iran (IR) and a two-digit number.
