ISO 3166-2:SC

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:SC is an ISO standard which defines geocodes. It is the subset of ISO 3166-2 that applies to Seychelles. Each two-part code consists of the ISO 3166-1 code for Seychelles (SC) and a two-digit number.

[edit] Newsletters
