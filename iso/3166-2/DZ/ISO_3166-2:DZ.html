ISO 3166-2:DZ

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:DZ is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Algeria. It covers the country's 48 provinces.

The first part of each code is the ISO 3166-1 code DZ for Algeria (Dzayar), the second part is the provincial code. (The names of the provinces alphabetically)
