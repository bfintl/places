ISO 3166-2:GW

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:GW is an ISO standard which defines geocodes. It is the subset of ISO 3166-2 that applies to Guinea-Bissau. Each two-part code consists of the ISO 3166-1 code for Guinea-Bissau (GW) and a two-digit number.

[edit] Codes

The codes define eight regions (regiões) and one autonomous sector (sector autónomo).

GW-BA Bafata
GW-BM Biombo
GW-BS Bissau*
GW-BL Bolama
GW-CA Cacheu
GW-GA Gabu
GW-OI Oio
GW-QU Quinara
GW-TU Tombali

* autonomous sector
