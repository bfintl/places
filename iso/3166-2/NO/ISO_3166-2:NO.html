ISO 3166-2:NO

From Wikipedia, the free encyclopedia

Jump to: navigation, search

ISO 3166-2:NO is an ISO standard which defines geocodes: it is the subset of ISO 3166-2 which applies to Norway. The code covers 19 counties (fylke). The first part of the code is the ISO 3166-1 country code NO for Norway, the second part is two-digit-numeric (with leading zeros).
